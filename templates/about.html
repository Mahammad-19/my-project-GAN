About This Project

Medical Image Synthesis Using GANs — BCA Final Year Project

Project Overview

This project implements a Generative Adversarial Network (GAN) to synthesise realistic pulmonary chest X-ray images. The system addresses the critical challenge of limited, imbalanced medical imaging datasets — a common barrier in training accurate AI diagnostic models.

By learning statistical patterns from real chest X-rays, the GAN produces synthetic radiographs that preserve key anatomical structures — lung fields, rib patterns, clavicles, and soft-tissue details — without containing any real patient data.
